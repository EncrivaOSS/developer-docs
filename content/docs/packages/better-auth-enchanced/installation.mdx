## Package Installation
<p>Choose your preferred package manager to install Better-Auth Enhanced:</p>

<Tabs items={['npm', 'yarn', 'pnpm', 'bun']} groupId="package-manager">
  <Tab value="npm">
  ```bash
  npm install @encriva/better-auth-enchanced
  ```
  </Tab>
  <Tab value="yarn">
  ```bash
  yarn add @encriva/better-auth-enchanced
  ```
  </Tab>
  <Tab value="pnpm">
  ```bash
  pnpm add @encriva/better-auth-enchanced
  ```
  </Tab>
  <Tab value="bun">
	```bash
	bun add @encriva/better-auth-enchanced
	```
  </Tab>
</Tabs>
